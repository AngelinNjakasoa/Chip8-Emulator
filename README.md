# Chip-8 Emulator

Yet another Chip8 emulator and tools.

Work in progress

| Tool | Description |
|:-----|-----------:|
|c8-disassembler|Translates chip8 opcode to chip8 assembly language|

## Requirements

 - cmake 3.9
 - SDL2

## Build

$ git clone https://github.com/AngelinNjakasoa/Chip8-Emulator.git </br>
$ cd Chip8-Emulator  </br>
$ mkdir build  </br>
$ cd build  </br>
$ cmake ..  </br>
$ make </br>
