# Chip-8 Emulator

Chip8 emulator from scratch

...Work in progress...

## Build

$ git clone https://github.com/AngelinNjakasoa/Chip8-Emulator.git </br>
$ cd Chip8-Emulator  </br>
$ mkdir build  </br>
$ cd build  </br>
$ cmake ..  </br>
$ make </br>
